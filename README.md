###How to run EvtGen external decayer class in STAR(RCF)  

```bash
mkdir TEST
cd TEST
git clone https://github.com/XiaozhiBai/StarEvtGen.git
cd StarEvtGen
starver DEV
source setup.csh
./cons.csh
```
